# Guides

Step-by-step guides written as instructions for AI agents and developers.

This directory contains prescriptive, task-oriented documentation designed to help developers and AI coding agents perform specific tasks with avaje-simple-logger.

## Available Guides

- [Add avaje-simple-logger to an Existing Maven Project](./add-avaje-simple-logger-to-maven-project.md) - Install avaje-simple-logger, replace Logback or Log4j 2, and configure JSON or plain output.
- [Add MDC, Fluent Key/Value, and OpenTelemetry Context to Logs](./add-structured-context-to-logs.md) - Use MDC, SLF4J 2 fluent `addKeyValue()`, and OpenTelemetry trace correlation in both JSON and plain formats.
- [Add AWS AppConfig to Your Project](./add-aws-appconfig-to-project.md) - Use AWS AppConfig with avaje-config to update `log.level.*` dynamically at runtime.

## Adding Guides

Create new markdown files in this directory for common tasks such as:
- Setup and configuration
- Integration with specific frameworks
- Testing approaches
- Troubleshooting
- Best practices

Each guide should:
1. Have a clear title and purpose
2. Start with prerequisites
3. Provide step-by-step instructions
4. Include code examples
5. Mention common issues and solutions
