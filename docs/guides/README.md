# Guides

Step-by-step guides written as instructions for AI agents and developers.

This directory contains prescriptive, task-oriented documentation designed to help developers and AI coding agents perform specific tasks with avaje-simple-logger.

## Adding Guides

Create new markdown files in this directory for common tasks such as:
- Setup and configuration
- Integration with specific frameworks
- Testing approaches
- Troubleshooting
- Best practices

Each guide should:
1. Have a clear title and purpose
2. Start with prerequisites
3. Provide step-by-step instructions
4. Include code examples
5. Mention common issues and solutions
